# Patchflow Blackmarket

Simpelt og moderne blackmarket system til FiveM.

## Features

- Clean moderne UI med Mantine + React
- Fuld localization support (EN, DA)
- Logging system (ox_lib logger + Discord webhooks)
- ESX & QBX framework support
- Type-safe Lua med LuaLS annotations

## Reference Resource

Det her er designet som en reference resource der demonstrerer moderne practices:

- Proper utilization af Lua Language Server
- Type definitions og annotations
- Clean config structure
- Brug af enums og constants
- Struktureret logging system

## UI Source

UI source code er ikke inkluderet med vilje. Vi holder på vores components og hooks som vi har lavet.
